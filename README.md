# Solar System Explorer

A premium dark-themed single page application for exploring the Solar System.

## Features

- Premium dark aesthetic with deep space background
- Interactive planet carousel with distinctive accent colors
- Dynamic data display showing planet stats (Mass, Gravity, Length of Day)
- Background stardust particle effect
- Smooth transitions and hover effects
- Responsive design for all devices

## Technologies Used

- Vanilla HTML, CSS, and JavaScript
- No external dependencies
- Fully responsive design

## Project Structure

```
solar-system-explorer/
├── index.html          # Main application entry point
├── style.css           # All styling for the application
├── app.js              # JavaScript logic and interactivity
└── README.md           # This file
```

## How to Run

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Explore the Solar System!

## Design Highlights

- Deep space background with animated stardust particles
- Each planet has its own distinctive accent color
- Smooth hover effects and transitions
- Responsive layout that works on mobile and desktop
- Detailed information panel when clicking on planets
